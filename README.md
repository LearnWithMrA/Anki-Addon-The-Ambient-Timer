# Ambient Review Timer

Two subtle circular clocks that appear in the corners of the Anki reviewer while you study.

- The **left clock** shows how long you took to answer that card last time
- The **right clock** counts up while you think
- Both disappear when you flip the card

The clocks are intentionally faint so they give you awareness of your pace without adding pressure.

## Installation

Copy the `ambient_timer` folder into your Anki addons directory and restart Anki.

| Platform | Path |
|----------|------|
| Mac | `~/Library/Application Support/Anki2/addons21/` |
| Windows | `%APPDATA%\Anki2\addons21\` |
| Linux | `~/.local/share/Anki2/addons21/` |

Or install directly via **Tools > Add-ons > Install from file** using the `.ankiaddon` file.

## Options

Go to **Tools > Ambient Timer Options** to adjust:

- Position: top or bottom corners
- Clock size, number size, opacity
- Show or hide tick marks and labels
- Clock colour with automatic light/dark mode adjustment
- Colour progression: the live clock changes colour as you pass your Easy, Good, Hard, and Again time thresholds

Toggle the addon on/off via **Tools > Ambient Timer: ON / OFF**.

## Requirements

Anki 2.1.45+
